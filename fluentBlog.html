Short app description

1. Introduction

The application was built using the Next.js technology due to the speed and ease of development, efficiency and the ability to render the code on the server side, which contributes to better SEO.

Below is quickly view of app:

In the case of application styling, a utility-first CSS framework called tailwind css was used, which enables efficient, fast and easy styling. In addition to the framework, custom styles have been added to the more custom parts of the application.

All styles and packages in Next.js applications are imported in the main file named _app. It is a convention that has been adopted by the creators. Below is an example of the structure of styles in the project
